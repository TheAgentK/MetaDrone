Detailed Description

Provides the access to a request to the HttpListener.

The HttpListenerRequest class cannot be inherited.

object  state 
)

Begins getting the client's X.509 v.3 certificate asynchronously.

This asynchronous operation must be completed by calling the EndGetClientCertificate method. Typically, that method is invoked by the requestCallback delegate.

Returns
An IAsyncResult that contains the status of the asynchronous operation.
Parameters
requestCallbackAn AsyncCallback delegate that references the method(s) called when the asynchronous operation completes.
stateAn object that contains a user defined object to pass to the requestCallback delegate.
Exceptions
NotImplementedExceptionThis method isn't implemented.
