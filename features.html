How does it work?

Control

Using the Meta 1, we have created an augmented reality viewer for the wearer to make controlling the quadcopter as natural as possible.

Processing

The user's inputs in 3D space are processed in Unity, and sent through WebSockets and Node to the quadcopter.

Flight

Using autonomous flight libraries, the quadcopter interperts the commands in physical space.
